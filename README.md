# Lyrix

### Vue.js/VueX web-app for loading lyrics

This is a private educational project I've created to learn Vue.js & VueX.
Feel free to fork and use it.

To install dependencies:

```bash
$ npm install
```

To develop:

```bash
$ npm run dev
```

To build for production:

```bash
$ npm run build
```

To lint your code:

```bash
$ npm run lint
```

To run tests:

```bash
$ npm test
```
